# 炭少年攤位互動色塊示意圖

這是一個使用 Next.js 和 React 開發的互動式色塊編輯器，用於設計和調整攤位配置。

## 功能特點

- 顏色編輯模式：可以選擇並修改各個區塊的顏色
- 區塊編輯模式：可以添加新的區塊並自定義其屬性
- 支援矩形和圓形兩種形狀
- 即時預覽效果
- 可刪除非基本結構區塊
- 一鍵重置功能

## 開發環境設置

```bash
# 安裝依賴
npm install

# 運行開發服務器
npm run dev

# 建置生產版本
npm run build

# 運行生產版本
npm run start
```

## 技術棧

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
